# 圖片對應字幕時間軸

這版字幕是依照 `圖片/` 資料夾裡的素材重新寫的，比原本的通用字幕更貼近畫面。

建議在 CapCut 使用：

- 字幕檔：`outputs/subtitles_by_image_capcut_utf8.srt`
- 影片長度：60 秒
- 比例：16:9

## 建議剪輯順序

| 時間 | 建議圖片 | 字幕 |
|---|---|---|
| 0-5s | `系所簡介.png` | 南大資工，以前瞻務實為方向，培養資訊工程專業人才。 |
| 5-10s | `3大領域學程.png` | 課程規劃涵蓋雲端網路、智慧計算與應用資訊。 |
| 10-15s | `同學實作電路.jpg` | 從課堂到實作，學生親手把理論接成電路。 |
| 15-20s | `實作電路板.jpg` 或 `實作電路板2.jpg` | 每一條線路，都是觀察、測試與修正的過程。 |
| 20-25s | `實驗室儀器.jpg` | 在實驗室中，儀器幫助我們看見系統的變化。 |
| 25-32s | `實驗室視波器.jpg` 或 `實驗室視波器2.jpg` | 透過波形量測，學習判斷訊號是否符合設計。 |
| 32-38s | `實驗室視波器3.jpg` 或 `實驗室視波器4.jpg` | 從結果中找問題，再把問題一步步解決。 |
| 38-44s | `畢業專題展.JPG` | 到了專題階段，學生把想法做成可以展示的成果。 |
| 44-50s | `畢業專題同學講解.JPG` 或 `畢業專題同學講解2.JPG` | 也練習把技術、設計與應用，清楚說給別人聽。 |
| 50-54s | `畢業專題上台報告.JPG` 或 `畢業專題上台報告2.JPG` | 上台報告，讓專業能力被整理、表達與交流。 |
| 54-57s | `資工系期中系大會報告.JPG` | 系上活動與分享，連結學習、研究與未來選擇。 |
| 57-60s | `期中系大會2.JPG` 或片尾文字卡 | 國立臺南大學資訊工程學系，走向下一個可能。 |

## CapCut 小調整

- `系所簡介.png` 與 `3大領域學程.png` 文字很多，建議用裁切或縮放，把重點區域放大。
- 電路板和示波器照片可加慢慢放大，讓畫面看起來像在觀察細節。
- 專題展與上台報告照片適合左右平移，讓觀眾看到人、海報和簡報畫面。
- 片尾可以另外做一張黑底或深藍底文字卡，放「國立臺南大學資訊工程學系」。

## Zero-paid-token Declaration

This project can be completed without buying commercial LLM tokens, using a personal paid API key, subscribing to AI video generation services, or requiring high-end hardware. The agent workflow is demonstrated through OpenClaw-style mock/stub agents, structured JSON handoff files, and an execution trace. The final video was edited in CapCut using user-provided images and subtitles.
